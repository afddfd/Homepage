VS Code 설정

VS Code 웹에서 사용하기 VS Code 홈페이지 바로 다운로드

Windows에서 Dev-C++을 컴파일러 포함 옵션으로 설치하여 사용한다고 가정합니다. 컴파일러를 추가 설치하지 않고 이미 설치된 컴파일러를 재사용할 것입니다.
디스크 공간 약 1 GB가 사용됩니다.

사전 작업 - OS 설정
sysdm.cpl - 고급 - 환경 변수 - 시스템 변수 - PATH에 C:\Program Files (x86)\Embarcadero\Dev-Cpp\TDM-GCC-64\bin 추가
언어 변경
Ctrl+Shift+P - Configure Display Language 검색해서 한국어 선택, 재시작.
기본 환경설정
Ctrl+, Font Size 28 Font Family "Consolas, 나눔고딕코딩, 'Courier New', monospace"
확장 프로그램 설치 Ctrl+Shift+X - 아래 다섯 개 확장 프로그램 설치
C/C++ / C/C++ Extension Pack / Live Preview / BrainFuck Syntax - Live Preview는 HTML 확인용
폴더 설정과 C 컴파일러 설정
적당히 폴더를 선택하고 "예, 작성자를 신뢰합니다." 선택
Ctrl+Shift+P - C/C++ 구성 선택 - 구성 편집(UI)
대부분은 건드릴 것이 없으나, 여기서 컴파일러 경로와 커스텀 헤더파일 설정, IntelliSense 모드, C/C++ 표준을 설정 가능.
.vscode 폴더 내에 tasks.json 파일을 만들고 https://bit.ly/1158-5 내용 적당히 붙여넣기.
지금은 C++를 다루지 않으므로 C++ 컴파일 부분은 주석 처리.
바로 가기 설정. Ctrl+K Ctrl+S, 우측 상단 "바로 가기 키 열기"
내용을 이것으로 변경
// 키 바인딩을 이 파일에 넣어서 기본값을 덮어씁니다.
[
//컴파일
{ "key": "F9", "command": "workbench.action.tasks.build" },

//실행
{ "key": "F10", "command": "workbench.action.tasks.test" }
]

Ctrl+Shift+P 터미널: 기본 프로필 선택 - cmd.exe

이제 Dev-C++과 동일하게 F9을 눌러 컴파일, F10을 눌러 실행 하시면 됩니다.
2023. 5. 25. 추가 - Python 구축법
Python.org 에서 CPython 다운로드 후, VS Code "Python" Extension 설치하면 실행이 가능 합니다.
keybindings.json(Ctrl+K Ctrl+S...) 에 {"key": "shift+f9","command": "python.execInTerminal"} 입력 후 저장하면 Shift+F9으로 Python 프로그램 실행이 가능해집니다.
참고 - https://webnautes.tistory.com/1158
기타 단축키 - Ctrl+K Ctrl+T Theme Settings, Ctrl+Shift+C Open Terminal
